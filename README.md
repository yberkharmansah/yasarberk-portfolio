# Yaşar Berk Harmanşah - Portföy Sitesi

Ben Kimim?  
Yazılımcı, oyun geliştirici ve AI meraklısı olarak yaptığım işleri burada topluyorum.

## Bölümler
- Hakkımda
- Projeler
- İletişim (LinkedIn, GitHub, mail)

## Teknolojiler
- Vue.js
- Bootstrap 5
- GitHub Pages / Netlify (yayın için)

## Kurulum
```bash
npm install
npm run dev
```

## Ortam Değişkenleri
Uygulama ve seed script için gerekli environment değişkenlerini `.env` dosyanıza ekleyin.

```bash
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
VITE_FIREBASE_MEASUREMENT_ID=...
VITE_CLOUDINARY_CLOUD_NAME=...
VITE_CLOUDINARY_UPLOAD_PRESET=...

FIREBASE_API_KEY=...
FIREBASE_AUTH_DOMAIN=...
FIREBASE_PROJECT_ID=...
FIREBASE_STORAGE_BUCKET=...
FIREBASE_MESSAGING_SENDER_ID=...
FIREBASE_APP_ID=...
FIREBASE_MEASUREMENT_ID=...
```

## Seed (Opsiyonel)
```bash
npm run seed-db
```
